#커뮤니티 게시글 제목 분석 #CA(Community Analysis)

프로젝트 설명

"CA"는 크롤링 기능을 사용하여 각종 커뮤니티에 올라오는 게시글의 제목을 분석하여 사용자에게 제공해주는 사이트입니다.
또한 형태소 분석 결과를 통한 추천검색어 기능과 검색단어에 대한 게시글을 분석해주는 기능이 있습니다.

적용기술

- 프레임워크 : Spring Framework
- 데이터베이스 : MySQL, MongoDB
- 기타 : R, Rserve, KoNLP, Jsoup, jQuery

#국가자료종합목록 Open API를 사용한 도서 검색 사이트 #책장

프로젝트 설명

"책장"은 국가자료종합목록 Open API를 사용하여 도서관에 비치된 책을 검색하여 제공해주는 사이트입니다.
사용자의 검색 기록을 바탕으로 추천 도서 목록을 제공합니다.

적용기술

- 프레임워크 : Spring Framework
- 데이터베이스 : Oracle 11g
- 적용 공공데이터(OpenAPI) : 국가자료종합목록(KOLIS-NET) Open API
- 기타 : kakao 지도 API, 네이버 SmartEditor2
